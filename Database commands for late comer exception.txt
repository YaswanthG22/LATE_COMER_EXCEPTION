‚Üí You should enter SQL> mode.



Step 3: Open the Pluggable Database (PDB)

Oracle 21c uses pluggable databases.
Run these commands:

SHOW PDBS;


You‚Äôll see something like:

CON_ID CON_NAME   OPEN MODE
3      ORCLPDB     MOUNTED


Now open it:

Switch to it:

ALTER SESSION SET CONTAINER = ORCLPDB;


Step 4: Create Your Own User

Run this:

CREATE USER employee_user IDENTIFIED BY emp123;


If you get ‚Äúinvalid common user or role name,‚Äù you forgot to set the container (redo Step 3).

Now grant permissions:

GRANT CONNECT, RESOURCE, CREATE SESSION, CREATE TABLE TO employee_user;
ALTER USER employee_user QUOTA UNLIMITED ON USERS;


‚úÖ This creates a normal database user with table privileges.
